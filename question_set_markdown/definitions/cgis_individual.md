# CGI-S — Clinical Global Impression Severity (Individual/Self-rated Adapted)

**Domain:** Mental Health (Global)  
**Population:** Adults; CESPHN individual questionnaire

## Description
Adapted CGI-S for self-report context in CESPHN program. Same 7-point severity anchor as clinician-rated version. See clinical_global_impressionseverity_scale.md for full detail.

## Scoring & Clinical Interpretation

- Self-rated CGI correlates with clinician CGI but may underestimate severity in psychosis and overestimate in depression.
- **≥5** = markedly or more severely ill → clinical review before next scheduled appointment.
