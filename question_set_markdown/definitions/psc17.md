# PSC-17 — Pediatric Symptom Checklist (17-item)

**Domain:** Pediatric Symptom Checklist  
**Population:** Children 4–16; parent-rated

## Description
17-item parent-rated screening tool with 3 subscales: Internalising (5 items), Attention (7 items), and Externalising (5 items). Items scored 0–2 (Never / Sometimes / Often).

## Scoring & Clinical Interpretation

| Subscale | Positive Screen Cut-off | Clinical Action |
|---|---|---|
| Internalising | ≥5 | Anxiety/depression assessment |
| Attention | ≥7 | ADHD assessment |
| Externalising | ≥7 | Conduct/ODD assessment |
| Total | ≥15 | Full mental health assessment |

- Any positive subscale = refer for disorder-specific assessment.
- Total ≥15 = psychosocial impairment present → referral recommended.
