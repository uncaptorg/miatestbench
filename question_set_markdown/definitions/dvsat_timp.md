# DVSAT — Domestic Violence Safety Assessment Tool (TIMP)

**Domain:** Domestic and Family Violence  
**Population:** Adults; TIMP program

## Description
Structured domestic violence safety assessment tool used in the TIMP program context. Assesses lethality risk factors, frequency and severity of violence, and protective factors.

## Scoring & Clinical Interpretation

- **Any identified DV risk** = mandatory safety planning protocol activation.
- High lethality indicators (e.g., strangulation, weapon use, threats to kill, isolation) = immediate safety response.
- Clinician uses tool to determine whether statutory DV service referral and/or mandatory reporting is required.

## Source
Adapted from validated DV assessment frameworks for TIMP program use.
