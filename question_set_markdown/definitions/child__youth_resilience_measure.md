# CYRM — Child and Youth Resilience Measure

**Domain:** Resilience  
**Population:** Children and youth (5–18)

## Description
Multi-dimensional resilience measure assessing strengths across individual (social skills, peer relationships, empathy, humour), relational (caregiver/family), and community (school, culture) levels. Multiple versions for different age groups.

## Scoring & Clinical Interpretation

- Higher total score = more resilience resources available.
- No single clinical cut-off — subscale profile used to identify where resilience is most depleted.

| Low-scoring Domain | Clinical Implication | Intervention |
|---|---|---|
| Individual | Poor peer skills, low empathy | Social skills groups; narrative therapy |
| Relational | Disrupted attachment or family | Family therapy; carer support |
| Community | School disengagement, cultural disconnection | School liaison; cultural connection programs |

- Use results to build strengths-based intervention plan — "what's strong, not just what's wrong."
