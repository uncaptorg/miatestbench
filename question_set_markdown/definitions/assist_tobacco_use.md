# ASSIST — Tobacco Involvement

**Domain:** Tobacco Use  
**Population:** Adults (16+)

## Description
WHO ASSIST tobacco module covering smoking frequency, craving, and quit attempts.

## Scoring & Clinical Interpretation

| Score | Risk Level | Clinical Action |
|---|---|---|
| 0–3 | Low | Brief advice |
| 4–26 | Moderate | NRT/Cessation brief intervention |
| ≥27 | High | Intensive cessation support; pharmacotherapy |

- Current smoking + psychotropic medication = metabolic monitoring priority.
