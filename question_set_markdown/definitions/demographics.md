# Demographics — CESPHN

**Domain:** Demographics  
**Population:** Adults; CESPHN program

## Description
Standard demographic data collection including age, gender identity, cultural background, Aboriginal and Torres Strait Islander status, country of birth, language spoken at home, postcode, and referral source.

## Scoring & Clinical Interpretation

Not scored. Informs:
- Cultural formulation and culturally responsive care
- Social determinants of health (housing, financial stress, isolation)
- Service access and equity monitoring
- ATSI status — specific mental health screening and care pathways apply

## Source
CESPHN demographic data standards.
