# Overall Anxiety Severity and Impairment Scale (OASIS)

**Domain:** Anxiety  
**Population:** Adults; transdiagnostic

## Description
Full-title version of OASIS. 5-item transdiagnostic anxiety severity and impairment scale. See oasis.md for full detail.

## Scoring & Clinical Interpretation

| Score | Severity | Clinical Action |
|---|---|---|
| 0–7 | Subclinical | Monitor |
| 8–11 | Mild | Brief intervention |
| 12–15 | Moderate | CBT referral |
| 16–20 | Severe–Very Severe | Active/urgent treatment |

- **≥8** = positive screen.
