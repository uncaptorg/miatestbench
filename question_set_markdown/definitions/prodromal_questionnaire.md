# Prodromal Questionnaire (Full version)

**Domain:** Psychosis-Like Experiences  
**Population:** Adults and young people (14+)

## Description
Full-title version of the PQ-16 prodromal questionnaire. 16-item self-report screen for attenuated psychotic symptoms. See pq16.md for full detail.

## Scoring & Clinical Interpretation

- **≥6 endorsed items** = positive screen → refer to early psychosis service.
- Distress weighting improves specificity.
- Requires follow-up with CAARMS or SIPS clinical interview.
