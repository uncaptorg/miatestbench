# BMC Psychosis Screen

**Domain:** Psychosis-Like Experiences  
**Population:** Adults; BMC program

## Description
Brief in-house psychosis screen covering hallucinations (auditory, visual), delusions (persecutory, referential, grandiose), thought disorder, and disorganisation. Used as an initial flag within BMC program workflows.

## Scoring & Clinical Interpretation

No composite score — item-level review:

| Item Endorsed | Clinical Action |
|---|---|
| Auditory hallucinations | Clinical interview for psychosis; risk assess |
| Persecutory delusions | Clinical interview; assess insight |
| Thought disorder | Cognitive and functional assessment |
| Any endorsed item | Do not initiate psychostimulants without psychiatric review |

- Any endorsed item = refer for structured psychiatric interview before progressing treatment.
- Lack of distress with endorsed items may indicate established psychosis with reduced insight.
