# EQ-5D-Y — EuroQol Youth (UNIQ Individual)

**Domain:** Overall Health  
**Population:** Young people; UNIQ program configuration

## Description
EQ-5D-Y as used in the UNIQ test program questionnaire. Same instrument as eq5dy_individual.md.

## Scoring & Clinical Interpretation

- Same 5-dimension + VAS scoring as standard EQ-5D-Y.
- Utility <0.7 = moderate health impairment.
