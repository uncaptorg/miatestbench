# Brief NSSI Assessment Tool

**Domain:** Non-Suicidal Self-Injury  
**Population:** Adolescents and adults

## Description
Structured brief assessment of non-suicidal self-injury (NSSI) covering method(s), frequency, age of onset, function (emotion regulation, self-punishment, anti-dissociation, communication), and level of medical injury.

## Scoring & Clinical Interpretation

No composite score — clinician reviews:

| Factor | Clinical Significance |
|---|---|
| Multiple methods | Higher severity and chronicity |
| Daily/weekly frequency | Requires immediate intervention |
| Medical tissue damage | Physical safety review required |
| Suicidal function endorsed | Reclassify as suicidal behaviour |
| Onset in childhood | Trauma-informed formulation priority |

- NSSI **without** suicidal intent still requires safety planning and function-matched intervention (DBT skills for emotion regulation function; trauma therapy for self-punishment function).
- Any NSSI with medical injury requires same-day medical review.
