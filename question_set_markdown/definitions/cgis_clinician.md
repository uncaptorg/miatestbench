# CGI-S — Clinical Global Impression Severity (Clinician)

**Domain:** Mental Health (Global)  
**Population:** All ages; clinician-rated

## Description
Standard clinician-rated CGI-S as used in the Mental Health (new) program questionnaire. See clinical_global_impressionseverity_scale.md for full detail.

## Scoring & Clinical Interpretation

- 1–7 scale; **≥5** = markedly ill → review treatment plan; **7** = emergency response.
- Pair with CGI-I at follow-up to track treatment response.
