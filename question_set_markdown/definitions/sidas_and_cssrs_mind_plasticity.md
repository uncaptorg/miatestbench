# SIDAS + C-SSRS — Combined Suicidality Assessment (Mind Plasticity)

**Domain:** Suicidality  
**Population:** Adults; Mind Plasticity program

## Description
Same combined SIDAS + C-SSRS instrument as sidas_and_cssrs.md, configured for the Mind Plasticity program context.

## Scoring & Clinical Interpretation

- **SIDAS:** 0–20 Low; 21–44 Moderate; **≥45 High risk**.
- **C-SSRS:** Any ideation with intent/plan or any behaviour = emergency action.
- See sidas_and_cssrs.md for full detail.
