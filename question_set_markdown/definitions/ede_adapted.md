# EDE Adapted — Eating Disorder Examination (CESPHN Adapted Version)

**Domain:** Eating Behaviours and Body Image  
**Population:** Adults; CESPHN program

## Description
Adapted version of the EDE-Q for the CESPHN eating behaviours and body image program. Same four subscale structure as the full EDE-Q. See eating_disorder_examination_questionnaire.md for full detail.

## Scoring & Clinical Interpretation

- **Global score ≥4** = clinical range.
- Shape/Weight Concern ≥4 + purging/compensatory behaviour = medical review same session.
- All scores require contextualisation with weight history and physical health findings.
