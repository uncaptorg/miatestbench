# QIDS — Quick Inventory of Depressive Symptomatology

**Domain:** Depressed Mood  
**Population:** Adults

## Description
Same instrument as qids.md — full-title version used in specific program configurations. 16-item self-report covering all DSM-5 MDD symptom domains.

## Scoring & Clinical Interpretation

| Score | Severity | Clinical Action |
|---|---|---|
| 0–5 | None | Monitor |
| 6–10 | Mild | Self-management |
| 11–15 | Moderate | Active treatment |
| 16–20 | Severe | Intensive treatment |
| 21–27 | Very Severe | Urgent review |

- **≤5** = remission; ≥50% drop = response.
