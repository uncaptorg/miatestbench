# RCADS — Revised Children's Anxiety and Depression Scale (Parent-report)

**Domain:** Anxiety and Depression (Youth)  
**Population:** Children and adolescents 8–18; parent-rated

## Description
Parent-rated version of RCADS. Same 47-item structure across 6 DSM-aligned subscales. See rcads_child.md for full detail.

## Scoring & Clinical Interpretation

- T-scores ≥70 on any subscale = clinical range → assessment and treatment.
- Parent–child discordance is clinically informative: child underreporting may indicate internalising disorder with shame; parent overreporting may reflect carer anxiety.
