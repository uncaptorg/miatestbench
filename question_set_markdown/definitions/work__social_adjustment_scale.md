# Work and Social Adjustment Scale (WSAS) — Full Title Version

**Domain:** Social and Occupational Function  
**Population:** Adults

## Description
Full-title version of WSAS. Same 5-item measure of functional impairment. See wsas.md for full detail.

## Scoring & Clinical Interpretation

- Total 0–40. **>20** = significant impairment.
- Used as primary functional outcome measure alongside symptom scales.
