# MAAS — Mindful Attention Awareness Scale

**Domain:** Mindfulness and Meditation  
**Population:** Adults and adolescents (14+)

## Description
15-item self-report measuring dispositional mindfulness — the frequency with which individuals experience present-moment attention in daily life. Scored 1–6 per item (1 = Almost always; 6 = Almost never), averaged.

## Scoring & Clinical Interpretation

| Average Score | Mindfulness Level | Clinical Note |
|---|---|---|
| 1.0–2.5 | Low mindful awareness | Priority for mindfulness-based intervention |
| 2.5–3.5 | Below average | Mindfulness skills development recommended |
| 3.5–4.5 | Average (general population ~4.0) | Maintain; integrate into wellbeing plan |
| 4.5–6.0 | High mindful awareness | Protective factor; reinforce practice |

- **Low MAAS** is associated with higher depression, anxiety, and chronic pain scores.
- Used as primary outcome measure in MBSR/MBCT programmes — expect 0.5–1.0 point increase with 8-week course.
- Adolescent norms slightly lower than adult norms.
