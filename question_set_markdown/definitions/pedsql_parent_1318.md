# PedsQL — Parent Report (Ages 13–18)

**Domain:** Quality of Life  
**Population:** Adolescents 13–18 years; parent-rated

## Description
Parent proxy-report PedsQL for adolescent age group. See pedsql_child_1318.md for scoring detail.

## Scoring & Clinical Interpretation

- <70 total = at-risk; <55 = significant impairment.
- Parent-teen discordance in this age group is common — use to spark collaborative discussion rather than adjudicate "correct" score.
