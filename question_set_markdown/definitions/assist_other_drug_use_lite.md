# ASSIST Lite — Other Drug Use (Brief)

**Domain:** Other Substance Use  
**Population:** Adults

## Description
Shortened ASSIST screen for non-alcohol, non-tobacco, non-cannabis substance use.

## Scoring & Clinical Interpretation

- Moderate = brief intervention; High = specialist referral.
- Any stimulant use (methamphetamine, cocaine) + psychotic symptoms = urgent psychiatric review.
