# EQ-5D-Y — EuroQol Youth (Support/Carer-rated)

**Domain:** Overall Health  
**Population:** Children and young people; carer-rated

## Description
Carer or support person-rated EQ-5D-Y. Useful for younger children or where self-report is unreliable.

## Scoring & Clinical Interpretation

- Same 5-dimension + VAS structure. Large carer–child discrepancy = clinical discussion warranted.
- Carer VAS <60 = significant perceived health impairment → priority case review.
