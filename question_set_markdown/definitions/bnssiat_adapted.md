# BNSSIAT Adapted — Brief Negative Symptom Scale / Social Interaction Anxiety

**Domain:** Social Interaction / Negative Symptoms  
**Population:** Adults; CESPHN program

## Description
Adapted scale measuring negative symptoms of psychosis (avolition, social withdrawal, flat affect) and/or social interaction anxiety, depending on program configuration.

## Scoring & Clinical Interpretation

- Higher scores = greater negative symptom burden or social anxiety.
- Elevated negative symptoms in psychosis predict poor functional recovery — prioritise psychosocial rehabilitation.
- Distinguish from depression (both present with avolition and withdrawal) — course and context differentiate.

## Source
CESPHN adapted tool — based on BNSS (Brief Negative Symptom Scale) principles.
