# Cognitive Behaviour Assessment (UBWell)

**Domain:** Cognition  
**Population:** Adults; UBwell program

## Description
Assessment of cognitive behaviour patterns including rumination, worry, avoidance, and behavioural activation levels used in the UBwell wellbeing programme.

## Scoring & Clinical Interpretation

Not scored on a composite scale. Item-level review identifies:
- High rumination = cognitive defusion techniques; behavioural activation
- High worry + low behavioural activation = GAD pathway; ACT/CBT
- Avoidance across multiple domains = exposure hierarchy development

## Source
UBwell program-specific measure based on CBT assessment framework.
