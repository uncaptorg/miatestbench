# Clinical Stage — Clinician Rating (McGorry Staging)

**Domain:** Clinical Stage  
**Population:** All mental health patients; clinician-rated

## Description
Clinician-rated staging model based on Patrick McGorry's clinical staging framework for mental illness, analogous to cancer staging. Stages 0 through 4 describe illness progression from pre-illness risk through first episode to persistent severe disorder.

## Scoring & Clinical Interpretation

| Stage | Description | Intervention |
|---|---|---|
| 0 | Increased risk; no current symptoms | Prevention; monitoring |
| 1a | Mild non-specific symptoms | Early intervention; watchful waiting |
| 1b | Ultra-high risk; subthreshold symptoms | Indicated prevention; cognitive therapy |
| 2 | First episode of disorder | Evidence-based treatment for specific disorder |
| 3a | Incomplete remission after first episode | Treatment intensification |
| 3b | Recurrence or relapse | Long-term treatment; relapse prevention |
| 3c | Multiple recurrences | Maintenance treatment; psychosocial rehabilitation |
| 4 | Severe, persistent, unremitting disorder | Specialist case management; intensive support |

- Stage determines intervention intensity and setting (primary vs. secondary vs. tertiary care).
- Staging should be reviewed regularly — recovery and stage reduction are possible.
