# PC-PTSD-5 + PCL-C — Combined PTSD Screen

**Domain:** Post-Traumatic Stress  
**Population:** Adults; OpenArms program

## Description
Combined administration of the PC-PTSD-5 initial screen with the PCL Civilian version. Used in the OpenArms veteran and first responder program context.

## Scoring & Clinical Interpretation

- **PC-PTSD-5 ≥3** = positive screen → full PCL-5 warranted.
- **PCL-C** (civilian version, 17 items): **≥44** = probable PTSD in civilian populations.
- High arousal cluster scores (hypervigilance, exaggerated startle) = occupational impairment risk in first responders.
- Combine with trauma history to assess chronicity and complexity.
