# OCI-CV5 — OCD Inventory Current Version 5

**Domain:** Obsessive-Compulsive Disorder  
**Population:** Adults (16+)

## Description
Self-report measure of OCD symptom frequency and distress. Covers washing, checking, ordering, obsessing, hoarding, and neutralising subscales. Current version 5 reflects updated factor structure.

## Scoring & Clinical Interpretation

| Total Score | Severity | Clinical Action |
|---|---|---|
| 0–21 | Subclinical | Monitor; psychoeducation |
| 21–40 | Mild | CBT/ERP self-help or guided |
| 41–60 | Moderate | ERP with therapist; consider SSRI |
| ≥61 | Severe | Intensive ERP; combined treatment; specialist review |

- **Subscale interpretation:** Highest-scoring subscale = primary ERP target.
- Hoarding subscale ≥10 = specialised hoarding disorder treatment pathway.
- **OCI-R cut-off ≥21** widely used for clinical range in research.
