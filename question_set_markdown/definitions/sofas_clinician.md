# SOFAS — Social and Occupational Functioning Assessment Scale (Clinician-rated)

**Domain:** Everyday Function  
**Population:** Adults; clinician-rated

## Description
Clinician-rated version of SOFAS. 0–100 scale assessing functional ability in work, social, and personal domains, assessed independently of psychiatric symptom severity. See sofas_individual.md for full scoring detail.

## Scoring & Clinical Interpretation

- **<60** = functional disability → rehabilitation plan.
- Use alongside symptom measures — functional recovery often lags symptom improvement.
- A SOFAS rise of 10+ points over an episode = clinically meaningful functional improvement.

## Source
APA DSM-IV Appendix B. [psychiatry.org](https://www.psychiatry.org/psychiatrists/practice/dsm)
