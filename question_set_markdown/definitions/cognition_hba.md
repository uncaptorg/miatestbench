# Cognition Assessment (HBA / Ageing)

**Domain:** Cognition  
**Population:** Older adults; HBA/Kildare ageing program

## Description
Cognitive screening items covering subjective memory complaints, concentration difficulties, word-finding problems, and disorientation. Adapted for use in older adult mental health settings.

## Scoring & Clinical Interpretation

Not scored on a composite scale. Clinical flags:
- **Multiple endorsed domains** = formal cognitive assessment (MoCA, MMSE) required
- **Rapid onset cognitive change** = urgent medical review (delirium, stroke, infection)
- **Gradual cognitive decline + depression** = differentiate between dementia and pseudodementia (treat depression first, reassess)

## Source
Standard cognitive screening framework; MoCA available at [mocatest.org](https://www.mocatest.org/)
