# SOFAS — Social and Occupational Functioning Assessment Scale (Support/Carer-rated)

**Domain:** Everyday Function  
**Population:** Adults; support person/carer-rated version

## Description
Carer or support person-rated SOFAS. Particularly useful when client self-report is unreliable (e.g., psychosis, severe depression with poor insight). See sofas_individual.md for scoring detail.

## Scoring & Clinical Interpretation

- Large discrepancy between client and carer SOFAS ratings = explore insight and collateral history.
- Carer-rated SOFAS <50 = significant observed functional impairment → case review.

## Source
APA DSM-IV Appendix B. [psychiatry.org](https://www.psychiatry.org/psychiatrists/practice/dsm)
