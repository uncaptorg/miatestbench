# EQ-5D-Y — EuroQol Youth Health Status (Individual)

**Domain:** Overall Health  
**Population:** Children and young people (8–15); self-rated

## Description
5-dimension health status measure plus Visual Analogue Scale (VAS). Dimensions: mobility, self-care, usual activities, pain/discomfort, anxiety/depression. Youth version uses simplified language. Each dimension rated on 3-level scale (no / some / extreme problems).

## Scoring & Clinical Interpretation

| VAS Score | Health State | Clinical Note |
|---|---|---|
| 80–100 | Good–Excellent | Routine |
| 60–79 | Moderate | Identify impaired dimension(s) |
| <60 | Poor | Active intervention; dimension-specific referral |

- Utility index derived from dimension profile using country-specific value sets (0 = death; 1 = perfect health; some states < 0).
- **Utility <0.7** = moderate health impairment across population norms.
- Dimension-level analysis identifies priority areas (e.g., anxiety/depression = mental health referral; pain = medical review).
