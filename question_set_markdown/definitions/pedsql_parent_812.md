# PedsQL — Parent Report (Ages 8–12)

**Domain:** Quality of Life  
**Population:** Children 8–12 years; parent-rated

## Description
Parent proxy-report PedsQL for 8–12 age group. See pedsql_child_812.md for scoring detail.

## Scoring & Clinical Interpretation

- Same thresholds: <70 at-risk; <55 significantly impaired.
- High parent-child discordance → explore insight, carer burden, or different informant perspectives.
