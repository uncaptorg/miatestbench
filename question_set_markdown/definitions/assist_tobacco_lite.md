# ASSIST Lite — Tobacco Use (CESPHN Adapted)

**Domain:** Tobacco Use  
**Population:** Adults; CESPHN program

## Description
Shortened tobacco ASSIST screen used in CESPHN initial questionnaire.

## Scoring & Clinical Interpretation

- Moderate/High = cessation referral and pharmacotherapy consideration.
- Smoking + antipsychotic use = dose adjustment may be needed on cessation (smoking induces CYP1A2).
