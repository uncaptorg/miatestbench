# Current Need — Clinician Rating

**Domain:** Current Need  
**Population:** All mental health patients; clinician-rated

## Description
Clinician assessment of the client's current level of required mental health support and service intensity. Used to match clients to appropriate care tier and guide step-up/step-down decisions.

## Scoring & Clinical Interpretation

| Need Level | Description | Action |
|---|---|---|
| Self-management | Client manages independently with minimal support | Discharge planning; self-help resources |
| Primary care | GP-level support sufficient | GP Mental Health Care Plan |
| Secondary care | Specialist outpatient support required | Referral to community mental health |
| Tertiary/Intensive | Acute/crisis or residential level of care needed | Step-up; crisis team; inpatient |

- Current need is dynamic — reassess at every review.
- Mismatch between clinical stage and current need = explore barriers to engagement or treatment.

## Source
Australian mental health service framework — no single published instrument source.
