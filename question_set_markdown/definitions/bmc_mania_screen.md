# BMC Mania Screen (ASRM-based)

**Domain:** Mania-Like Experiences  
**Population:** Adults; NEAMI program

## Description
Mania screen as deployed in the NEAMI initial questionnaire, based on ASRM principles. Assesses elevated/irritable mood, increased energy, decreased sleep need, racing thoughts, and distractibility.

## Scoring & Clinical Interpretation

- **≥6 on ASRM component** = probable manic/hypomanic episode → urgent review.
- In bipolar disorder context: any score increase from baseline warrants medication review.
- Elevated score + risk-taking behaviour = immediate safety assessment.
