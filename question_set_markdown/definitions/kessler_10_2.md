# Kessler Psychological Distress Scale (10-item) — Version 2

**Domain:** Psychological Distress  
**Population:** Adults (18+)

## Description
K10 as configured for the Kildare program (version 2). Identical instrument to kessler_10.md. See k10.md for full detail.

## Scoring & Clinical Interpretation

| Score | Likely Status | Clinical Action |
|---|---|---|
| 10–19 | Likely well | No intervention |
| 20–24 | Likely mild disorder | Brief intervention |
| 25–29 | Likely moderate disorder | Referral |
| 30–50 | Likely severe disorder | Urgent review |
