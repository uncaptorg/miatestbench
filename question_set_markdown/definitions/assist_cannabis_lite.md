# ASSIST Lite — Cannabis Use (Brief)

**Domain:** Cannabis Use  
**Population:** Adults; CESPHN program

## Description
Shortened ASSIST cannabis screen for use in initial questionnaire workflows.

## Scoring & Clinical Interpretation

- **Moderate/High involvement** = brief intervention or specialist referral.
- Cannabis use in young people + psychotic symptoms = urgent psychiatric review.
