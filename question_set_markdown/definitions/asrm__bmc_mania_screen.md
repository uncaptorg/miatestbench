# ASRM — Altman Self-Rating Mania Scale (BMC/CESPHN version)

**Domain:** Mania-Like Experiences  
**Population:** Adults; BMC/CESPHN program

## Description
ASRM as used in the BMC/CESPHN initial questionnaire. Same 5-item self-report measuring manic symptoms over the past week. See altman_selfrating_mania_scale.md for full detail.

## Scoring & Clinical Interpretation

- **≥6** = probable hypomanic/manic episode → urgent clinical review.
- Assess medication adherence, sleep pattern, and recent stressors.
