# Sleep Battery for Older Adults — PSQI + ISI + STOP

**Domain:** Sleep  
**Population:** Older adults (65+); Kildare program

## Description
Combined sleep assessment battery: Pittsburgh Sleep Quality Index (PSQI), Insomnia Severity Index (ISI), and STOP (obstructive sleep apnoea screen). Addresses the three most common sleep disorders in older adults.

## Scoring & Clinical Interpretation

**PSQI:** >5 = poor sleep → sleep hygiene and further assessment.

**ISI:**

| Score | Severity | Clinical Action |
|---|---|---|
| 0–7 | No insomnia | Monitor |
| 8–14 | Subthreshold insomnia | CBT-I self-help or guided |
| 15–21 | Moderate insomnia | CBT-I (first line); consider short-term hypnotic |
| 22–28 | Severe insomnia | Specialist review; intensive CBT-I |

**STOP (≥2 "Yes"):** High OSA risk → polysomnography referral.

- In older adults: benzodiazepine use for sleep = falls risk — always flag.
- Disrupted sleep + cognitive symptoms = screen for dementia.
