# ASSIST — Other Substance Use (Excluding Cannabis)

**Domain:** Other Substance Use  
**Population:** Adults; Kildare program

## Description
ASSIST other substance use module excluding cannabis (assessed separately). Used where cannabis module is administered independently.

## Scoring & Clinical Interpretation

- Same Low/Moderate/High framework as full ASSIST.
- Any injecting = automatic high risk.
