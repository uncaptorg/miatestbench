# PedsQL — Parent Report (Ages 5–7)

**Domain:** Quality of Life  
**Population:** Children 5–7 years; parent-rated

## Description
Parent proxy-report PedsQL for young children. 21 items covering physical, emotional, social, and school/pre-school functioning.

## Scoring & Clinical Interpretation

- **<70 total** = at-risk; **<55** = significant impairment.
- For children who cannot self-report, parent PedsQL is the primary QoL outcome measure.
- Discrepancy between parent and child report (in older children) = clinically informative about carer burden.
