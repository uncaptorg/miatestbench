# AUDIT-C Short — Alcohol Use (Brief Screen)

**Domain:** Alcohol Use  
**Population:** Adults; Kildare program

## Description
Shortened AUDIT-C configuration used in the Kildare program initial questionnaire. Three-item alcohol consumption screen.

## Scoring & Clinical Interpretation

- **≥3 (women) or ≥4 (men)** = positive screen for hazardous alcohol use.
- Positive screen → full AUDIT or brief motivational intervention.
