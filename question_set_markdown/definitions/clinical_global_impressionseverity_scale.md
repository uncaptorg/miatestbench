# CGI-S — Clinical Global Impression – Severity Scale

**Domain:** Mental Health (Global)  
**Population:** All ages; clinician-rated

## Description
Single-item 7-point clinician-rated global severity scale. Clinician rates severity relative to all patients with the same diagnosis they have ever seen. Used across all psychiatric conditions. The companion CGI-Improvement (CGI-I) rates change from baseline on the same scale.

## Scoring & Clinical Interpretation

| Score | Severity | Clinical Action |
|---|---|---|
| 1 | Normal; not at all ill | No clinical action |
| 2 | Borderline mentally ill | Monitor |
| 3 | Mildly ill | Psychotherapy; low-intensity intervention |
| 4 | Moderately ill | Active treatment (therapy + possible medication) |
| 5 | Markedly ill | Active treatment; frequent review |
| 6 | Severely ill | Intensive treatment; consider inpatient |
| 7 | Most extremely ill | Emergency care |

- **CGI-I 1–2** (Very much / Much improved) = treatment response.
- **CGI-I 3** (Minimally improved) = partial response; review treatment plan.
- Despite subjectivity, CGI shows excellent criterion validity across diagnoses.
