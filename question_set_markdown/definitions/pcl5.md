# PCL-5 — PTSD Checklist for DSM-5

**Domain:** Post-Traumatic Stress  
**Population:** Adults (18+)

## Description
20-item self-report aligned to all four DSM-5 PTSD symptom clusters: intrusion (B), avoidance (C), negative alterations in cognition/mood (D), and arousal/reactivity (E). Scored 0–4 per item (Not at all → Extremely).

## Scoring & Clinical Interpretation

| Score | Interpretation | Clinical Action |
|---|---|---|
| 0–32 | Below threshold | Watchful waiting if trauma history present |
| 33–80 | Probable PTSD | Full diagnostic interview; trauma-focused treatment |

- **≥33** = provisional PTSD diagnosis (general mental health settings).
- **Cluster B ≥1 + C ≥1 + D ≥2 + E ≥2** = DSM-5 symptom criterion approach.
- **≥10-point reduction** = clinically significant change; **≤10–20** = good end-state functioning.
- Distinguish from complex PTSD — high D cluster (negative cognitions/mood) suggests more pervasive impact.
