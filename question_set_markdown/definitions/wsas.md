# WSAS — Work and Social Adjustment Scale

**Domain:** Social and Occupational Function  
**Population:** Adults (18+)

## Description
5-item self-report measuring functional impairment across work/study, home management, social leisure, private leisure, and close relationships. Each item 0–8 (0 = no impairment; 8 = very severe impairment). Total 0–40.

## Scoring & Clinical Interpretation

| Score | Severity | Clinical Action |
|---|---|---|
| 0–10 | Subclinical | No action |
| 10–20 | Significant but suggests mild–moderate disorder | Functional goals in treatment plan |
| >20 | Moderate to severe impairment | Active rehabilitation; consider step-up care |

- **>20** = significant impairment consistent with moderately severe disorder.
- Individual item analysis identifies the most impaired domain for targeted intervention (e.g., high work item → vocational rehabilitation; high relationship item → couples/family therapy).
