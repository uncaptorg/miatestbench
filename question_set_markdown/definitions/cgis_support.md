# CGI-S — Clinical Global Impression Severity (Support/Carer)

**Domain:** Mental Health (Global)  
**Population:** All ages; carer-rated version

## Description
Carer-rated version of CGI-S. Useful when patient insight is limited. See clinical_global_impressionseverity_scale.md for full scoring.

## Scoring & Clinical Interpretation

- Carer rating ≥5 = markedly ill in carer's perception → review care plan.
- Large carer–patient discrepancy = explore insight and carer burden.
