# SDQ — Strengths and Difficulties Questionnaire (Parent/Teacher 4–10)

**Domain:** Child Mental Health  
**Population:** Children 4–10 years; parent or teacher-rated

## Description
25-item questionnaire with 5 subscales (5 items each): Emotional Symptoms, Conduct Problems, Hyperactivity/Inattention, Peer Problems, and Prosocial Behaviour. Each item 0–2 (Not true / Somewhat true / Certainly true). Impact supplement assesses impairment.

## Scoring & Clinical Interpretation

**Total Difficulties Score (Emotional + Conduct + Hyperactivity + Peer, max 40):**

| Score | Band | Clinical Action |
|---|---|---|
| 0–13 | Normal | No concern |
| 14–16 | Borderline | Monitor; reassess in 3 months |
| ≥17 | Abnormal | Referral for assessment |

- **Impact supplement ≥2 (parent) or ≥3 (teacher)** = significant functional impairment → priority referral.
- High Hyperactivity + high Conduct = ADHD/ODD assessment; high Emotional + high Peer = anxiety/depression assessment.
