# SOFAS — Social and Occupational Functioning Assessment Scale (Self-report)

**Domain:** Everyday Function  
**Population:** Adults; self-rated version

## Description
0–100 single-item rating of social and occupational functioning independent of psychiatric symptom severity. Self-rated version for use in CESPHN universal questionnaire.

## Scoring & Clinical Interpretation

| Score | Function Level | Clinical Action |
|---|---|---|
| 91–100 | Superior functioning | No concerns |
| 71–90 | Slight impairment | Monitor; psychoeducation |
| 51–70 | Moderate impairment | Functional rehabilitation; supported employment/education |
| 31–50 | Serious impairment | Intensive community support; care coordinator |
| 11–30 | Major impairment | High-level support; consider intensive case management |
| 1–10 | Unable to function | Crisis/acute care |

- **<60** = functional disability warranting active rehabilitation.
- Symptom severity and SOFAS often diverge — SOFAS tracks real-world recovery better than symptom scores alone.

APA DSM-IV Appendix B. [psychiatry.org](https://www.psychiatry.org/psychiatrists/practice/dsm)
