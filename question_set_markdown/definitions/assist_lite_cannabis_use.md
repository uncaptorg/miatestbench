# ASSIST Lite — Cannabis Use (Brief)

**Domain:** Cannabis Use  
**Population:** Adults

## Description
Shortened ASSIST cannabis screen. See assist_cannabis_use.md for full detail.

## Scoring & Clinical Interpretation

- Moderate = brief intervention; High = specialist referral.
- Adolescent cannabis use + psychotic symptoms = urgent psychiatric review.
