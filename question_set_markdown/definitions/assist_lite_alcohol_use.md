# ASSIST Lite — Alcohol Use (Brief)

**Domain:** Alcohol Use  
**Population:** Adults

## Description
Shortened ASSIST alcohol screen for rapid initial assessment.

## Scoring & Clinical Interpretation

- Low/Moderate/High risk classification. Moderate = brief intervention; High = specialist referral.
