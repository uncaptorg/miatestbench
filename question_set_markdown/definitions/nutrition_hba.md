# Nutrition Assessment (HBA / Ageing)

**Domain:** Nutrition  
**Population:** Older adults; HBA program

## Description
Structured nutritional assessment for older adults covering meal frequency, food variety, appetite changes, weight loss, and nutritional supplements.

## Scoring & Clinical Interpretation

- Unintentional weight loss ≥5% in 3 months = nutritional risk → dietitian referral.
- Poor appetite + depression = treat both concurrently (appetite often improves with depression treatment).
- B12, folate, and vitamin D deficiency common in older adults — flag for GP review.

## Source
Australian Dietary Guidelines for Older Australians. [eatforhealth.gov.au](https://www.eatforhealth.gov.au/)
